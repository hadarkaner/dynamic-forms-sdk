# Docs

A full documentation site (VitePress). To run it locally:

```bash
cd docs
npm install
npm run docs:dev
```

- [index.md](index.md) — home, who this is for.
- [get-started.md](get-started.md) — quick start.
- [how-to-use.md](how-to-use.md) — the SDK's API, including slots and offline handling.
- [examples.md](examples.md) — real code examples (browser, mobile, server).
- [implementation.md](implementation.md) — architectural explanation of every component, the data model, and the choice to run on Neon.
- [testing.md](testing.md) — a guide to testing the full flow end to end (UI and `curl`).
- [screenshots.md](screenshots.md), [video.md](video.md) — media.
